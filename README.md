# Sistema de Controle de Biblioteca

Projeto desenvolvido para a disciplina de Algoritmos e Programação.

## Descrição

O sistema permite o gerenciamento de livros de uma biblioteca por meio de um menu interativo no terminal.

Cada livro possui:

* Título
* Autor
* Ano de publicação
* Código (ID único)
* Status (DISPONÍVEL ou INDISPONÍVEL)

## Funcionalidades

* Cadastrar livro
* Consultar livro por código
* Consultar livros por autor
* Alterar dados de um livro
* Remover livro
* Listar todos os livros cadastrados
* Realizar empréstimo
* Realizar devolução
* Encerrar o programa

## Estrutura do Projeto

* `biblioteca.py` → arquivo principal com o menu do sistema.
* `funcoes.py` → funções de gerenciamento dos livros e definição da classe `Livro`.
* `testes` → arquivo com os dados de alguns livros

## Tecnologias Utilizadas

* Python

## Conceitos Aplicados

* Variáveis
* Estruturas condicionais (`if`, `else`)
* Estruturas de repetição (`for`, `while`)
* Listas
* Funções
* Classes (structs em Python)
* Modularização de código

## Como Executar

1. Certifique-se de ter o Python instalado.
2. Execute o arquivo principal:

```bash
python biblioteca.py
```

3. Utilize o menu exibido no terminal.
